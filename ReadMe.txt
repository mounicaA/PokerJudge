Required software: Java 1.7

*******************************
Instructions to Compile and Run
*******************************
1. Make this folder current directory using cd
2. javac PokerJudge.java
3. java PokerJudge 2HQS2D2S5H 5C3CASQH9H

********************************
Assumptions
********************************
1.No Jokers
2.Only one deck of cards are played
3.Ace is counted as either less than 2 or greater than K
